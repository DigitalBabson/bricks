## Implementation Details

Add an active-filter pills bar and "Clear all" button to the bottom of `BrickFilter.vue`. This provides visual feedback showing which filters are active and allows quick removal.

**Current state (after Tasks 4.2–4.3):** BrickFilter has keyword input and location list. No visual indicator of active filters beyond the input value and the highlighted list item.

**Target state (from XD wireframe):**

```
┌──────────────────────────────────────────────────┐
│ [Brick Inscription: sample keyword ×]  [Clear all] │
│ [Brick Location: Class Walk of 2019 ×]             │
└──────────────────────────────────────────────────┘
```

The pills bar sits below the keyword input and location list, inside the same `<form>` container. It is hidden when no filters are active.

**Pill rendering logic:**

```typescript
// Computed
get activeFilters(): Array<{ type: string; label: string; value: string }> {
  const filters = [];
  if (this.inscription) {
    filters.push({
      type: 'inscription',
      label: 'Brick Inscription',
      value: this.inscription,
    });
  }
  if (this.locationId) {
    const loc = this.locations.find(l => l.id === this.locationId);
    filters.push({
      type: 'location',
      label: 'Brick Location',
      value: loc?.name ?? this.locationId,
    });
  }
  return filters;
}
```

**Pill markup:**

```html
<!-- Section C: Active Filters — only renders when filters are active -->
<div v-if="activeFilters.length > 0" class="tw-mt-4 tw-flex tw-flex-wrap tw-items-center tw-gap-2">
  <span
    v-for="filter in activeFilters"
    :key="filter.type"
    class="
      tw-inline-flex tw-items-center tw-gap-1
      tw-bg-brickLightGreen tw-text-brickSummerNight
      tw-font-zilla tw-text-sm
      tw-px-3 tw-py-1 tw-rounded-full
    "
  >
    {{ filter.label }}: {{ filter.value }}
    <button
      type="button"
      class="tw-ml-1 tw-text-brickSummerNight hover:tw-text-black focus:tw-outline-none"
      :aria-label="`Remove ${filter.label} filter`"
      @click="removeFilter(filter.type)"
    >
      ×
    </button>
  </span>

  <button
    type="button"
    class="
      tw-font-oswald tw-text-white tw-text-sm tw-uppercase
      tw-border tw-border-white tw-px-3 tw-py-1 tw-rounded
      hover:tw-bg-white hover:tw-text-brickCourtyardGreen
      tw-transition-colors tw-duration-200
    "
    @click="$emit('clearAll')"
  >
    Clear all
  </button>
</div>
```

**Methods:**

```typescript
removeFilter(type: string) {
  if (type === 'inscription') {
    this.$emit('update:inscription', '');
  } else if (type === 'location') {
    this.$emit('update:locationId', '');
  }
},
```
