## Implementation Details

Create `src/components/AppFooter.vue` — a stateless component that mirrors the header's first line.

**Structure:**

```
┌─────────────────────────────────────────────────────────────────┐
│  BABSON COLLEGE                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**Markup outline:**

```html
<footer class="tw-bg-brickBabsonGreen tw-w-full tw-py-4 tw-px-6">
  <div class="tw-max-w-brickMWL tw-mx-auto">
    <p class="tw-font-oswald tw-text-white tw-uppercase tw-text-2xl tw-font-semibold">
      Babson College
    </p>
  </div>
</footer>
```

**Key points:**

- Full-width dark green (`tw-bg-brickBabsonGreen`) bar — identical background to the header.
- "BABSON COLLEGE" — large, bold, uppercase, white, Oswald. Same styling as the header's first line.
- No subheader, no links, no copyright line.
- Inner content padded and max-width constrained to match header alignment (`tw-max-w-brickMWL tw-mx-auto`).
- No props, no emits.
